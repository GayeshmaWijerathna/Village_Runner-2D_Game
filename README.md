# Village_Runner-2D_Game
  The 2D Web Game 

**[How to Play...]
    [+] Use Enter for Go Next Level
    [+] Use Space_Bar for Jump

  [Complete Levels...]
    [#] Score 310 for win 1st Level
    [#] Score 350 for win 2nd Level
    [#] Score 450 for win 3rd Level
    [#] Score 400 for win 4th Level**
